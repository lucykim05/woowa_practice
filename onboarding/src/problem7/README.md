## 🚀 기능 요구 사항

레벨 2의 팀 프로젝트 미션으로 SNS(Social Networking Service)를 만들고자 하는 팀이 있다. 팀에 속한 크루 중 평소 알고리즘에 관심이 많은 미스터코는 친구 추천 알고리즘을 구현하고자 아래와 같은 규칙을 세웠다.

- 사용자와 함께 아는 친구의 수 = 10점
- 사용자의 타임 라인에 방문한 횟수 = 1점

사용자 아이디 user와 친구 관계를 담은 이차원 배열 friends, 사용자 타임 라인 방문 기록 visitors가 매개변수로 주어질 때, 미스터코의 친구 추천 규칙에 따라 점수가 가장 높은 순으로 정렬하여 최대 5명을 return 하도록 solution 메서드를 완성하라. 이때 추천 점수가 0점인 경우 추천하지 않으며, 추천 점수가 같은 경우는 이름순으로 정렬한다.

### 제한사항

- user는 길이가 1 이상 30 이하인 문자열이다.
- friends는 길이가 1 이상 10,000 이하인 배열이다.
- friends의 각 원소는 길이가 2인 배열로 [아이디 A, 아이디 B] 순으로 들어있다.
  - A와 B는 친구라는 의미이다.
  - 아이디는 길이가 1 이상 30 이하인 문자열이다.
- visitors는 길이가 0 이상 10,000 이하인 배열이다.
- 사용자 아이디는 알파벳 소문자로만 이루어져 있다.
- 동일한 친구 관계가 중복해서 주어지지 않는다.
- 추천할 친구가 없는 경우는 주어지지 않는다.

## 설계

### 입력

1. 유저 이름
2. 친구 관계
3. 방문자

### 검증

1. 사용자 아이디 검증(길이, 알파벳 소문자)
2. 친구관계 검증(배열 길이, 각 배열 length 2, 중복 여부)
3. 방문객 검증(배열 길이)

### 친구찾기

1. user와 친구인 사람 찾기
2. 그 사람들과 친구인 사람 찾기(user와 친구인 사람끼리 친구여도 점수 추가됨)

### 점수 계산

1. map 생성
2. 친구의 친구들 배열을 받아서
3. 점수 배분
4. visitor도 점수 배분

### 결과 도출

1. 점수 내림차순 정렬
2. 0점 제외하고 고르기
3. 5명 초과면 점수 순으로 자름
4. 동점이면 이름 순

### 출력

저 방식 너무 불편하니 걍 result만 출력하도록

1. 친구들이 다 0점이면 error
2. 결과 arr로 출력

## 구현 체크리스트

### 입력

- [ ] 유저 이름 입력 받기
- [ ] 이름 검증 후 return
- [ ] 친구 관계 입력 받기
- [ ] 아이디 검증
- [ ] 친구관계 검증 후 return
- [ ] 방문객 입력 받기
- [ ] 방문객 아이디 검증
- [ ] 방문객 검증 후 return

### 검증

- [ ] 사용자 아이디 검증
  - [ ] 길이 1 이상 30 이하
  - [ ] 알파벳 소문자로 이루어짐
- [ ] 친구 관계 검증
  - [ ] 배열 길이 1 이상 10000 이하
  - [ ] 각 배열 안에 2개 요소 들어간 배열들 들어있음
  - [ ] 배열 중복여부 확인
- [ ] 방문객 검증
  - [ ] 배열 길이 0 이상 10000 이하

### 친구찾기

- [ ] user와 친구인 사람들 찾아서 배열로 정리
- [ ] 그 사람들과 친구인 사람 찾아서 배열로 정리(user 있는채로)

### 점수 계산

- [ ] 점수 계산용 map 생성
- [ ] 친구의 친구 배열 받아오기
- [ ] 각 배열 순환하며 점수 10점씩
  - [ ] 이 때 user는 제외
- [ ] visitor 배열 받아오기
- [ ] visitor에 1점씩 점수 추가

### 결과 도출

- [ ] map의 value가 0점이 아닌 사람 내림차순 정렬
- [ ] 5명 초과면 앞에서부터 5명 잘라서 배열에 key(이름) 넣기
- [ ] 5명 이하면 그냥 그대로
- [ ] 점수 동점이면 이름 순으로 정렬

### 출력

- [ ] 만약 결과가 없으면(다 0점이면) error
- [ ] 결과 받아와서 출력
