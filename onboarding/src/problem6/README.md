## 🚀 기능 요구 사항

우아한테크코스에서는 교육생(이하 크루) 간 소통 시 닉네임을 사용한다. 간혹 비슷한 닉네임을 정하는 경우가 있는데, 이러할 경우 소통할 때 혼란을 불러일으킬 수 있다.

혼란을 막기 위해 크루들의 닉네임 중 **같은 글자가 연속적으로 포함** 될 경우 해당 닉네임 사용을 제한하려 한다. 이를 위해 같은 글자가 연속적으로 포함되는 닉네임을 신청한 크루들에게 알려주는 시스템을 만들려고 한다.

신청받은 닉네임 중 **같은 글자가 연속적으로 포함** 되는 닉네임을 작성한 지원자의 이메일 목록을 return 하도록 solution 메서드를 완성하라.

## 구현할 기능 목록

### 입력

- [x] 몇 명 입력할건지
- [x] 해당 수 만큼 이메일, 이름 입력받기
- [x] 다 받았으면 입력 종료
- [x] 각 배열 return

### 검증

- [x] 크루 수
- [x] 닉네임 길이
- [x] 닉네임 한글만
- [x] 이메일 형식
- [x] 이메일 길이

### 메인 로직

- [x] 정리용 map 생성
- [x] 이름 배열을 돌며 2글자씩 나눔
- [x] 나눈걸 중복 없애고 map에 있는지 확인
- [x] map에 있으면 key+1, 없으면 새 객체 생성
- [x] key가 2 이상인 2글자 찾기
- [x] 이름 배열 돌며 해당 글자 포함된 이름 찾기
- [x] 이름 index값 구해서 email 정렬 후 return

### 출력

- [x] 시작 부분 출력
- [x] 입력받은 이메일과 이름 배열 출력
- [x] 중복되는 사람의 이메일 출력

## 구현

### 입력

입력과정에서 number 즉, 총 인원수를 받았을 때 바로 검증처리를 하는 것이 더 낫다고 판단하여 받자마자 바로 검증로직을 실행시켰고, 크루의 정보를 받은 후에도 똑같이 진행하였다.<br/>
인원수를 받은 후에는 입력과정을 해당 숫자만큼 반복시키고, 입력이 완료되면 완료되었다는 내용을 print하도록 하였다.
