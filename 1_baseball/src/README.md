## 숫자 야구

힌트를 통해 컴퓨터가 뽑은 세 개의 숫자를 정확히 맞추는 프로그램

## 구현 기능 목록

### Model

Computer.js

- [ ] constructor→ 숫자 뽑기 → 저장
- [ ] 입력으로 숫자 배열이 주어졌을 때, 컴퓨터의 숫자와 비교해서 결과 값을 String으로 리턴

parser.js

- [ ] 사용자 입력을 정해진 delimiter로 split하여 리턴

userGuessValidator.js

- [ ] 사용자 입력 검증 기능
  - [ ] 배열 요소가 3가지 인지
  - [ ] 배열 내부 값이 비었는지
  - [ ] 배열 내부 값이 1~9 사이의 숫자인지
  - [ ] 중복되는 숫자가 존재하는지

gameOverValidator.js

- [ ] 사용자 게임 종료 응답 검증 기능
  - [ ] 값이 비었는지
  - [ ] 값이 숫자인지
  - [ ] 1 또는 2 범위의 숫자인지

### View

input.js

- [ ] 사용자 숫자 입력 요청
- [ ] 사용자 게임 종료 시 응답 요청

output.js

- [ ] 결과 출력 요청
- [ ] 에러 메세지 출력 요청

### Controller

baseballController.js

- [ ] 숫자 야구 게임 시작 메세지 출력 요청
- [ ] 컴퓨터 클래스 선언 및 숫자 뽑기 요청

- [ ] 사용자 입력 요청

  - [ ] 사용자 입력 파싱 요청
  - [ ] 사용자 입력 검증 요청

- [ ] 컴퓨터 객체에 사용자 배열 비교 요청
- [ ] 비교 결과 출력 요청
  - [ ] 만약 3스트라이크가 아닐 시 사용자 입력 반복
  - [ ] 3스트라이크일 시 게임 종료 메세지 출력
  - [ ] 사용자 게임 종료 응답 요청
    - [ ] 사용자 게임 종료 응답 검증 요청
      - [ ] 만약 사용자의 응답이 1일 시 게임 재시작
      - [ ] 만약 사용자의 응답이 2일 시 게임 종료
