# README.md

## 로또

사용자가 로또를 1000원 단위로 자동 구입하고 당첨 번호와 보너스 번호를 입력하여 당첨 통계를 출력하는 프로그램

## 프로젝트 구조

파일구조 첨부

## 설계

플로우차트 첨부
