## 설계

프로그램 진행 방향을 정리하면 다음과 같다.

1. 문자열 입력
2. 커스텀 구분자 파싱
3. 구분자를 바탕으로 문자열 처리
4. 계산 후 결과 출력

역할 분리에 따라 다음과 같이 나누게 되었다.
